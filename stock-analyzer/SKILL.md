# Stock Analyzer

## 使用方式

```bash
# 分析股票
python {baseDir}/scripts/analyze.py 00700.HK

# 或 A股
python {baseDir}/scripts/analyze.py 000001.SZ
```

## 功能

- 获取实时股价
- 技术指标分析
- 趋势判断
