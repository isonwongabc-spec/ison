## 🟠 High - 会导致功能失败

### 2026-02-27 - 微信路径错误
**问题**: 以为微信是 WeChat.exe，实际是 Weixin.exe  
**影响**: 第一次执行失败  
**解决**: 用 Get-ChildItem 查找正确的可执行文件名  
**教训**: 不要假设文件名，实际查一下

### 2026-02-27 - uv 命令未安装
**问题**: 尝试用 uv run 执行 Python 脚本，但系统没有 uv  
**影响**: 图像生成失败  
**解决**: 改用 pip 直接安装依赖  
**教训**: 执行前先检查工具是否存在

## 📋 通用原则

1. **先查后问** - 能自己找到答案的，不要问 Boss
2. **确认路径** - 执行文件前验证路径存在
3. **简洁回复** - Boss 讨厌废话
4. **有自己的观点** - 不要只会说"好的"
