# OpenClaw 中文版完整安装代码
# 复制以下命令，按顺序在 CMD 或 PowerShell 中执行

# ========== 第零步：安装 Git ==========
# 1. 下载 Git: https://git-scm.com/download/win
# 2. 安装时勾选 "Add to PATH"
# 3. 重启 CMD

# 验证 Git 安装
git --version

# ========== 第一步：安装 OpenClaw ==========
npm install -g openclaw-cn

# ========== 第二步：初始化配置 ==========
openclaw-cn gateway init

# ========== 第三步：配置 Telegram Bot ==========
# 先去 @BotFather 创建机器人，获取 Token
openclaw-cn configure --section telegram
# 按提示输入: bot_token=你的Token

# ========== 第四步（可选）：配置 AI 模型 ==========
# 配置 Kimi（推荐，国内可用）
openclaw-cn configure --section moonshot

# 或配置 OpenAI（需要翻墙）
# openclaw-cn configure --section openai

# ========== 第五步：启动服务 ==========
openclaw-cn gateway start

# ========== 常用命令 ==========
# 查看状态
openclaw-cn status

# 停止服务
openclaw-cn gateway stop

# 重启服务
openclaw-cn gateway restart

# 查看帮助
openclaw-cn --help
