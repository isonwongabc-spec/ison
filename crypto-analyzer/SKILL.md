# Crypto Analyzer

## 使用方式

```bash
# 分析 BTC
python {baseDir}/scripts/crypto.py BTC

# 分析 ETH
python {baseDir}/scripts/crypto.py ETH

# 同时分析多个币种
python {baseDir}/scripts/crypto.py BTC ETH SOL
```

## 支持币种

BTC, ETH, SOL, ADA, DOGE, XRP, DOT, AVAX 等
